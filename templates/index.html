Control Reference

Detailed guidance for the main controls, grouped the same way they appear in the sidebar.

Image Upload

Choose File

Upload the source image or PDF you want to trace. Drag-and-drop onto the upload area works the same as using the file picker.

Show Raw Image

Switches the preview between the untouched upload and the normalized image used for detection. Use it to compare whether preprocessing is helping or harming the trace.

Image Opacity

Fades the background image without changing the traced paths. Lower values make the magenta and blue overlays easier to inspect.

Detection Parameters

Dark Threshold

Controls how dark a pixel must be before it is treated as part of a line. Raise it to ignore faint background texture; lower it to recover lighter strokes.

Auto-Detect Threshold

Asks the app to estimate a useful threshold from the uploaded image. It is a starting point, not a final answer, so fine tuning may still be needed.

Normalization Mode

Controls whether upload preprocessing evens out lighting and contrast before detection. Use Auto for most images, Force On for uneven scans, and Force Off when the source is already clean.

Normalization Sensitivity

Sets how aggressively Auto mode decides to normalize. Higher sensitivity normalizes more images, which helps difficult scans but can flatten already-good artwork.

Merge Gap

Maximum endpoint distance for bridging broken skeleton paths. Increase it when lines have small interruptions; decrease it if unrelated features begin joining together.

Angle Priority

Biases merging toward endpoints that continue in a similar direction. Higher values prefer smoother continuation over pure nearest-gap bridging.

Min Path Length

Filters out short detected paths before optimization. Raise it to drop dots and tiny fragments; lower it only when genuine short strokes are being lost.

Auto-Detect Min Path Length

Lets the app suggest a minimum path length based on the current image. Useful when noise level changes a lot between jobs.

Optimisation

Tolerance

Sets the overall simplification looseness. Higher values allow bigger geometric shortcuts and usually produce fewer segments.

Simplification Strength

Controls how hard the optimizer tries to reduce vertices overall. It affects how aggressively the blue output departs from the raw magenta path.

Line Fit Strength

Controls how strongly the optimizer prefers long straight segments. Raise it when staircase-like lines should collapse to cleaner segments.

Short Path Protection

Preserves detail on short or jagged paths that would otherwise collapse too hard. Higher values keep more vertices on small features.

Score Preservation

Minimum retained path score for accepting a simplification. Higher values make the optimizer more conservative and less willing to reshape the path.

Enable Curve Fitting

Turns cubic fitting back on for SVG output. Leave it off while tuning line quality, then enable it if you want smoother bezier-style export.

Cubic Fit Tolerance

Controls how tightly cubic segments must follow the optimized path. Lower values preserve more shape detail, while higher values reduce cubic count.

Endpoint Tangent Strictness

Controls how strongly fitted cubic handles follow the local direction of the extracted path at the start and end. Higher values keep endpoints aligned more literally.

Force Orthogonal Paths As Lines

Keeps axis-aligned and corner-like paths as line segments instead of curves. Useful for technical drawings, grids, and anything that should stay crisp and geometric.

Output Options

Enable Path Optimisation

Toggles the blue-path optimization stage. When disabled, the app keeps the raw detected paths and forces the unoptimized layer to stay visible.

Include Unoptimised Paths

Adds the magenta pre-optimization paths to the SVG. It is mainly a debugging and comparison aid when checking what the optimizer changed.

Include Original Image

Embeds the source bitmap in the SVG export. Leave it on for review documents; turn it off for cleaner production vector output.

Actions

Process

Runs detection and starts optimization using the current settings. Use this when Auto is off or when you want a deliberate rerun after several changes.

Auto

Automatically reprocesses after parameter changes. Keep it on for quick iteration and turn it off when you want to batch several adjustments first.

Reset

Returns the session UI to its baseline state. Use it to clear the current job and undo exploratory tuning without reloading the page.

Stop Optimization

Stops a running optimization pass and keeps whatever progressive result is available. Useful when a job is taking too long or the intermediate result is already good enough.

SVG View And Export

Download SVG

Downloads the current SVG render to disk using the current output settings. This is the reliable export path on every browser the app supports.

Drag SVG File

Lets you drag the generated SVG out of the browser, but only on Chromium-family browsers where that drag payload is supported. On other browsers, download the SVG first and drag the saved file from your file manager.

Reset View

Resets pan and zoom in the SVG preview window. Use it after inspecting details closely so the whole result is framed again.
